
# AHAM Fashion - Web (Next.js) Starter

## Quick start
1. Create a Firebase project and enable Firestore and Authentication.
2. Create a Stripe account (test keys) and set environment variables.
3. Install dependencies:
   ```bash
   cd web
   npm install
   ```
4. Create a .env.local with the required variables (see earlier instructions).
5. Run the dev server:
   ```bash
   npm run dev
   ```
